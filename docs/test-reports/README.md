# Test Reports

Per-baseline test reports and traceability matrices, generated by CI from
Surefire results and the annotation scan (SDP §5). Naming:
`M<n>-report.md`, `M<n>-traceability.md`. Manual test verdicts (e.g.
SIM-TC-013 frontend smoke test) are appended to the milestone report with
date, name, and verdict. Files in this directory are generated or appended at
milestone gates only — never hand-edited afterwards.
